## Promise/A+ 规范的实现

### 安装
`npm i` 或者 `yarn` 安装依赖包

### 测试
依赖安装完毕后 `npm run test` 或者 `yarn test`。
<br>
还可以通 `npm run start` 或者 `yarn start` 启动一个测试服务器，在浏览器端进行测试。
